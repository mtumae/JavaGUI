<h3>SU Feeds</h3>

A simple rudimentary GUI that allows a user to register an account, make as well as view comments made by other students based on topics learnt in Strathmore University, Nairobi Kenya.
Built in:

<li>Java (Java Fx)</li>
<li>Postgresql</li>
